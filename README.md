# task-manager-fullstack

Sistema de Gerenciamento de Tarefas Full-Stack utilizando React + TypeScript (Frontend) e Node.js (Backend) - CRUD

# Tecnologias Utilizadas

# Frontend

- **React 18** - Biblioteca JavaScript para construção de interfaces
- **TypeScript** - Superset tipado do JavaScript
- **Vite** - Build tool rápida e moderna
- **Tailwind CSS** - Framework CSS utility-first
- **Axios** - Cliente HTTP para requisições à API

# Backend

- **Node.js** - Ambiente de execução JavaScript
- **Express** - Framework web minimalista
- **TypeScript** - Tipagem estática para maior segurança
- **CORS** - Middleware para controle de acesso
- **ts-node-dev** - Desenvolvimento com hot-reload

## Funcionalidades

- **Criar** novas tarefas com título e descrição
- **Listar** todas as tarefas cadastradas
- **Atualizar** status de conclusão das tarefas
- **Excluir** tarefas indesejadas
- Interface responsiva e moderna
- Feedback visual em tempo real
- API RESTful completa
